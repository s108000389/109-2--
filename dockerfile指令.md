### 查看現有 Docker Image
```
docker images
docker image ls
```
### 刪除 Docker Image
```
docker rmi image_id
docker rmi 名稱
強制刪除:docker rmi -f image_id
```
### 查看現有 Container
```
docker ps -a
```
### 停用及刪除 Container
```
docker stop container_id
docker rm container_id
```
### 停用及刪除所有 Container 
```
docker stop $(docker ps -a -q)
docker rm $(docker ps -a -q)
```
